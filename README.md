# Usage
![image](https://github.com/ogulcantopsakal/scene-navigator-unity/assets/36813238/705a0b9e-54c3-43f5-8089-6ee4b4ce0b68)

**Dropdown**


From here you can select all scenes in the project.


**S Button**


Used for single load.


**A Button**


Used for additive load.



**R Button**


It is used to refresh the scenes in the dropdown when we add or remove a new scene.


**+ Button**


It allows saving the selected scene as a preset.

![image](https://github.com/ogulcantopsakal/scene-navigator-unity/assets/36813238/0df2f3d1-bb13-40b1-a960-b5ccec0589ff)

**- Button**


Removes scene from presets.
